# Influencer Campaign Performance Tracker

## 1. Problem Statement

Influencer marketing has become an important strategy for promoting products and services through social media platforms. Brands collaborate with influencers to increase audience reach, engagement, and sales. However, it is difficult to identify which campaigns perform best without proper analysis.

The objective of this project is to analyze influencer marketing campaign data and build a dashboard that helps evaluate campaign effectiveness, audience engagement, and sales performance.

---

## 2. Dataset Description

The dataset contains information related to influencer marketing campaigns conducted across different platforms.

### Features Included

* Campaign ID
* Platform
* Influencer Category
* Campaign Type
* Start Date
* End Date
* Engagements
* Estimated Reach
* Product Sales
* Campaign Duration Days

The dataset was cleaned by handling missing values and removing duplicate records before analysis.

---

## 3. KPI Definitions

The following Key Performance Indicators (KPIs) were used:

### Total Campaigns

Total number of campaigns conducted.

### Total Reach

Total audience reached through all campaigns.

### Total Engagements

Total interactions generated from campaigns.

### Total Product Sales

Total products sold through influencer campaigns.

### Average Engagement Rate

Measures audience interaction effectiveness.

**Formula:**
Engagement Rate = (Engagements / Estimated Reach) × 100

### Average Campaign Duration

Average number of days a campaign runs.
